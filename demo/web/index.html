Overview

Avi Features

Local & Global Load Balancing

Intelligent L4–L7 load balancing with TLS termination, GSLB, DNS integration, and traffic steering.

Predictive Autoscaling

Analytics-driven automation dynamically scales Service Engines based on real-time demand.

Automation & Self-Service

API-first architecture enabling CI/CD, Infrastructure as Code, and self-service provisioning.

Integrations & Analytics

Full visibility with integrations into VMware Aria, Splunk, Ansible, Terraform, and cloud ecosystems.

Web Application Security

Built-in WAF, OWASP CRS protection, DDoS mitigation, rate limiting, and bot detection.

Kubernetes Ingress Services

Production-grade Kubernetes ingress with load balancing, security, observability, and traffic management.
